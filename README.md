# AILUU.CLO Store

Одностраничный интернет-магазин мусульманской женской одежды с корзиной, оформлением заказов и полноценным управлением через Telegram-бота.

## Что уже работает

- адаптивная одностраничная витрина;
- фильтры по категориям и карточки товаров;
- выбор размера, цвета и количества;
- корзина в браузере;
- форма заказа;
- уведомление о новом заказе в Telegram;
- добавление товара через Telegram с загрузкой фото;
- изменение названия, цены, описания, категории, размеров, цветов и фото;
- публикация/скрытие и отметка «Новинка»;
- удаление товара;
- просмотр заказов и изменение их статусов;
- изменение основных текстов сайта, контактов и валюты через Telegram;
- хранение данных и фотографий в подключаемой папке `data`;
- Docker и health endpoint.

## Быстрый запуск

1. Создайте Telegram-бота через `@BotFather` и получите токен.
2. Узнайте свой Telegram user ID.
3. Скопируйте настройки:

```bash
cp .env.example .env
```

4. Заполните `.env`:

```env
PORT=3000
PUBLIC_BASE_URL=https://your-domain.kg
TELEGRAM_BOT_TOKEN=123456:ABC...
TELEGRAM_ADMIN_IDS=123456789
ORDER_NOTIFICATION_CHAT_ID=123456789
```

5. Запустите:

```bash
docker compose up -d --build
```

Сайт откроется на `http://localhost:3000`. В Telegram отправьте боту `/start`.

## Запуск без Docker

Требуется Node.js 20 или новее. Внешние npm-зависимости не используются.

```bash
cp .env.example .env
npm start
```

## Команды Telegram

- `/start` — главное меню;
- `/add` — добавить товар;
- `/products` — список и редактирование товаров;
- `/orders` — последние заказы;
- `/settings` — тексты, контакты и валюта;
- `/cancel` — отменить текущее действие.

Доступ разрешён только ID из `TELEGRAM_ADMIN_IDS`. Не добавляйте токен бота в GitHub.

## Данные и резервное копирование

Рабочие данные создаются в `data/store.json`, фотографии — в `data/uploads/`. Эти файлы не коммитятся. Для резервной копии сохраняйте всю папку `data`.

## Развёртывание

Для production рекомендуется:

- домен и HTTPS через Nginx/Caddy;
- постоянный volume для `/app/data`;
- один экземпляр контейнера при использовании polling Telegram;
- ежедневная резервная копия папки `data`;
- секреты только в `.env` на сервере.
